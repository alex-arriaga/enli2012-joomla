# Aprendiendo Joomla! 3
Un ejemplo de un sitio en Joomla! 3 preparado especialmente para el **Encuentro Nacional de Software Libre**, edición 2012 en Puebla México: [Enli2012](http://www.enli.org.mx/).

## Notas importantes
- La base de datos de ejemplo se encuentra en **data/joomla3.sql**.
- El archivo **plantilla_periodico.tar.gz** corresponde al empaquetado de una plantilla de ejemplo para **Joomla! 3**.
- El archivo **plantilla-periodico-html.zip** es la misma plantilla sólo que en HTML5 puro.
